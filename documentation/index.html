Extras#back to top

Contact Form

The contact form will use php to send emails. Open the file sendmail.php located inside the php folder and change the example email found in the line 4 and your form is ready to use. Do Not delete the quote marks.

					     = "youremail@email.com"; //Recipient email, Replace with own email here

Newsletter with Mailchimp Integration

The Newsletter form in this template includes Mail Chimp integration, to integrate with your existing campaign just open your HTML file and find the newsletter form and replace the address by your own mail list address.

Boxed version

To use the boxed version simply use the class "boxed" in the body tag like in the example bellow:

						  < body id="page-top" class="boxed" >

Video Header

If you wish to use the video header version of the theme you should have a video in both .mp4 and .webm formats. You can use this website to convert the video. The videos should be placed inside the videos folder. The tag video poster will have the image that will be used as placeholder in case the video does not play.

Parallax

This theme uses Stellar and Skrollr for parallax effects, click in each link to read the documentation.

Map

To change the Open street Map section with your own location you can use This website to find your own latitude and longitude, then you can replace the longitude and latitude numbers in the file map.js located inside the js folder. 

						//Open street  Map
						var coord = [40.738270, -74.008911]; // <--- coordinates here

Icons

This template includes over 500 icons from FontAwesome 4.3 , Glyphicons and Flat Icons you can click in each link to find the icon map, to change the icons just replace with the icon name you want.

Fonts

The fonts used in this template are from Google Fonts, the fonts are Lato, and Baloo Thambi

Images

Image Performance

Logo
